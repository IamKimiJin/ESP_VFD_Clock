# ESP_VFD_Clock
ESP8266驱动的ИВ-18八位VFD时钟及B站粉丝数显

# 声明 
* 程序还属于测试版本 可能会有一些BUG
* 本项目需要用到的库 [wifi_link_tool](https://github.com/bilibilifmk/wifi_link_tool)  ArduinoJson 和time(以提供)

## 材料清单  
* ИВ-18 vfd *1
* ESP12F    *1
* IRF520N mos管 *1
* MAX6921 vfd驱动器 *1
* RCWL-0516 *1
* XL6009 升压模块 *1
* 微动 *1
* 二极管（仅防反） *1
### 瓶子的选择 
本项目中提供的3D模型基于以下尺寸   
* 瓶子高度21.1cm  
* 瓶内直径6cm
